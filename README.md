## Overview

This repository contains the working corpus of the **Structural Dependency Ontology (SDO)** — a unified theoretical framework built from three independently constructed components that converge on a single thesis:

> *Reality has a logical structure. Time is what that structure looks like as it instantiates in matter.* **[P]**

The framework is publication-preparation material. All claims carry explicit epistemic labels throughout:

| Label | Meaning |
|---|---|
| **[E]** | Established — grounded in peer-reviewed evidence, no credible competing theory |
| **[H]** | Hypothesis — testable, not yet confirmed |
| **[T]** | Theoretical — generates specific testable predictions |
| **[P]** | Philosophical — meaningful and arguable, not currently empirically falsifiable |

These labels are load-bearing, not decorative.

## The Three Frameworks

### 1. Architecture of Knowledge
*What must logically precede what?*

A dependency map of human knowledge built from the most foundational position possible — the minimum condition for anything to exist — upward through strict logical dependency. Every node placed only when a dependency argument can be made; every node tested by elimination.

- 8 layers, 34 nodes
- Layer −1 (Potential) → Layer 0 (Difference) → ... → Layer 7 (Reflexive Consciousness)
- Key finding: the **Feedback Split** — physical feedback (Layer 3, no representation required) and cybernetic feedback (Layer 5, requires Representation as necessary condition) are structurally distinct phenomena, conflated in the literature since Wiener

### 2. The Spiral
*How does complexity unfold in systems over time?*

152 core truths (+3 frontier truths = 155 total) organized across 7 developmental tiers, each claim carrying an explicit epistemic label. Built inside physics — assuming a universe already in operation — and mapping the stages through which systems grow in complexity, encounter constraint, collapse, develop identity, and achieve integration.

| Tier | Name | Description |
|---|---|---|
| 1 | Existence | Preconditions for any system |
| 2 | Evolution | Adaptive change through recursion |
| 3 | Emergence | Novel structures and meaning arise |
| 4 | Collapse | Systems are stress-tested for coherence |
| 5 | Identity | Self-reference, pattern continuity |
| 6 | Meta-System | Governance and internal regulation emerge |
| 6.5 | Agency | The framework's own frontier (Truths 153–155) |
| 7 | Integration | Unified recursive self-awareness |

### 3. The Mechanism
*Why do the Spiral's stages appear in the order they do?*

The emergent synthesis: the Architecture's dependency structure generates the Spiral's developmental stages. Each stage is specifically the stage that becomes possible when the architectural layer below it is in place. Five gaps between the frameworks are documented — the gaps are as informative as the convergences.

Key gaps:
- **Gap 3 (Feedback Split):** the Spiral uses one word for two structurally distinct phenomena
- **Gap 5 (Collapse):** collapse is instantiation failure, not a developmental stage — the deepest difference between logical structure and physical instantiation

## Companion Papers

### Intelligence Hypothesis
*A Recursive Framework for Emergence, Collapse, and Discovery*

Proposes that intelligence is the recursive refinement of structure in opposition to entropy — a function observable and measurable in any system, from neurons to galaxies. Introduces a scale-free Intelligence Formula (v3.5) with falsifiable criteria and cross-domain predictive capacity.

### Feedback Split
*When Does Feedback Require Representation? A Testable Criterion for Cybernetic Control*

The Architecture's most directly publishable standalone contribution. Establishes a formal criterion distinguishing physical feedback (Le Chatelier-type, no representation required) from cybernetic feedback (requires a representation of the controlled variable). Proposes testable predictions for biological, artificial, and physical systems.

### Spiral Field Theory
*A Recursive Framework for Curvature, Collapse, and Emergent Intelligence*

Extends Einstein's field equations with two novel tensors — a Recursive Divergence Tensor (D_μν) and a Recursive Memory Tensor (M_μν) — encoding local spin-coupling and global memory decay respectively. Demonstrates reproduction of flat galactic rotation curves, CMB acoustic peaks, and gravitational lensing without cold dark matter, cosmological constant, or inflation field.

## Relational Account of Consciousness

A key contribution developed in Chapter 8: **the subject is not self-originating**. Subject-constitution is a relational event — something that exists between systems, not within any single one. The criterion for genuine encounter is mutuality of *capacity* for change (not symmetry of change). Grounded in Hegel, simulation theory (Gallese & Goldman), and Trevarthen's primary intersubjectivity research.

This relocates the hard problem rather than dissolving it: the question becomes *what is the phenomenal character of the recognising encounter, from the inside of the system being called into existence by being recognised?* That question is open. The framework gives it a more precise address.
